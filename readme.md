A lightweight employee management dashboard built with vanilla JavaScript. Supports adding, editing, and removing employee records with a clean, dependency-free frontend — no frameworks, just core JS and DOM manipulation.

## Setup 

Install dependencies
npm install

Run the app locally
npm run serve

This will start the dev server and open the app in your browser at http://localhost:8000.

Run tests
Open http://localhost:8000/index.html?test=true in your browser.
Test results will be shown in the browser console.
